Why are chess moves used instead of passwords or PINs?

+

Chess moves deliver dynamic, unpredictable, personal patterns, vastly increasing key complexity and eliminating vulnerabilities common to static passwords or reused PINs.

What file types and sizes can CastleFile handle and is it compatible with mobile?

+

CastleFile supports any desktop file type or size—documents, images, videos, archives—and all processing is local. It works on mobile browsers, though very large files perform best on desktop.

How is CastleFile secure compared to other encryption tools?

+

By using client‑side AES‑GCM with SHA‑256‑derived keys from unique chess moves and storing no data or keys on servers, CastleFile delivers true zero‑knowledge privacy and tamper‑proof encryption.
